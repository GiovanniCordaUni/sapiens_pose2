# Sapiens Pose Estimation Pipeline (COCO17)

Pipeline per la stima della posa umana da video di esercizi di fisioterapia
basata su:
- YOLO (person detection)
- Meta Sapiens Pose (COCO 17 keypoints)

## Features
- Supporto singolo video o dataset (soggettoNNN/)
- Output JSONL + video overlay
- Configurazione via YAML
- Struttura modulare

## Note
I pesi dei modelli NON sono inclusi nel repository.
